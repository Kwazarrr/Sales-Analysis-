# Analiza sprzedaży

## Opis projektu
Projekt polega na analizie danych sprzedażowych w celu zrozumienia struktury przychodów oraz identyfikacji kluczowych produktów i kategorii.

## Cele
- Analiza przychodów w czasie
- Identyfikacja kluczowych kategorii
- Analiza produktów
- Zrozumienie źródeł przychodu

## Narzędzia
- Python
- pandas
- Jupyter Notebook

## Wnioski
- Przychody rosną, ale tempo wzrostu maleje
- Kategoria tech generuje około 80% przychodu
- Produkt B generuje największy przychód dzięki wysokiemu średniemu koszykowi
- Produkt A odpowiada za największą liczbę zamówień
